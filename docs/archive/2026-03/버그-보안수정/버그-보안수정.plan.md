## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 코드 분석 결과 Critical 5건·High 11건 포함 총 37건의 버그·보안 취약점이 발견됨. 일부는 현재 사용자에게 실시간 오류를 유발 중(리버튼 작성자 `maybeSingle` 오류, 무료 리버튼 익스플로잇 등) |

## 2. 기능 요구사항

### 우선순위 1 — Critical (즉시 수정)

#### 2-1. [SEC-01] Edge Function 인증 추가
- **대상**: `supabase/functions/settle-balance-games/index.ts`
- JWT 검증 추가: `Authorization` 헤더 필수화
- CORS를 `https://matbul.com`으로 제한 (현재 `*`)
- `platform/js/post.js` 호출부에 `session.access_token` 전달

#### 2-2. [BUG-06] `loadMyComment` 다중 행 오류
- **대상**: `platform/js/post.js:877-884`
- `.maybeSingle()` → `.is('parent_id', null)` 필터 추가로 루트 댓글만 조회
- 리버튼(대댓글)은 별도 쿼리로 처리

#### 2-3. [BUG-05] 리버튼 크레딧 원자성
- **대상**: `platform/js/post.js:1507-1512`
- 크레딧 차감(`spend_credits` RPC) → 성공 시에만 댓글 INSERT
- 실패 시 삽입된 댓글 롤백(DELETE)

#### 2-4. [BUG-03/04] `URL.createObjectURL` 메모리 누수
- **대상**: `platform/js/create.js:287, 368`
- 이전 object URL 추적 후 재렌더 시 `URL.revokeObjectURL()` 호출

#### 2-5. [BUG-01] `#commentList` 중복 클릭 리스너
- **대상**: `platform/js/post.js:1232, 1309`
- 두 리스너를 단일 리스너로 병합

---

### 우선순위 2 — High

#### 2-6. [SEC-02] LIKE 와일드카드 이스케이프
- **대상**: `platform/js/home.js:484`
- `%`, `_` 문자 이스케이프 후 `.ilike()` 전달

#### 2-7. [SEC-03] Edge Function CORS 제한
- **대상**: `supabase/functions/settle-balance-games/index.ts`, `delete-account/index.ts`
- `Access-Control-Allow-Origin: *` → `https://matbul.com`

#### 2-8. [SEC-04] `escapeHtml` 작은따옴표 미이스케이프
- **대상**: `platform/js/auth.js:120-123`
- `'` → `&#x27;` 추가

#### 2-9. [SEC-05] 크레딧 클라이언트 검증 TOCTOU
- **대상**: `platform/js/create.js:622-633`
- 클라이언트 잔액 체크 제거 또는 DB 트랜잭션 내 처리로 전환

#### 2-10. [BUG-07] `fmtNum` 포맷 문자열 `parseInt` 오류
- **대상**: `platform/js/profile.js:79`
- 팔로워 수를 `data-raw-count` 속성에 저장, 포맷 문자열 파싱 금지

#### 2-11. [BUG-09] `clearTimeout` 누락
- **대상**: `platform/js/create.js:631, 643`
- 크레딧 부족 early return 전 `clearTimeout(timeoutId)` 추가

#### 2-12. [BUG-08] 직렬 await → 병렬 처리
- **대상**: `platform/js/mypage.js:5-13`, `platform/js/profile.js:9-13`
- `Promise.all([])` 로 병렬화

#### 2-13. [BUG-21] `loadMyComment` 쿼리 필터
- BUG-06과 동일 파일, `.is('parent_id', null)` 추가로 통합 처리

---

### 우선순위 3 — Medium

#### 2-14. [BUG-11] 타이머 1s→60s 복귀 로직
- **대상**: `platform/js/home.js:946-948`
- 긴급 타이머 만료 후 60s 인터벌로 복귀 또는 타이머 정지

#### 2-15. [BUG-10] 게스트 `currentUser.id` null 참조
- **대상**: `platform/js/post.js:654`
- `if (myComment && currentUser)` 가드 추가

#### 2-16. [BUG-18/19] quiz.js 이모지 → SVG
- **대상**: `platform/js/quiz.js:48, 289`
- 이모지(`📭`, `📝`) 제거 → SVG 아이콘 대체

#### 2-17. [SEC-10] UUID 형식 검증
- **대상**: `platform/js/profile.js`
- `?id=` 파라미터 UUID 정규식 검증 추가
